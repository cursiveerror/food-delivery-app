## Про проєкт
Ctrl+Eat — це сучасний та швидкий вебдодаток для зручного замовлення та доставки їжі. Головна мета проєкту — надати користувачам максимально простий, інтуїтивно зрозумілий та естетичний інтерфейс для вибору страв без зайвого візуального шуму. 

Додаток розроблено у форматі Single Page Application (SPA), що забезпечує миттєвий перехід між сторінками без перезавантаження браузера, створюючи відчуття користування нативним застосунком.

## Можливості для користувачів

* **Динамічне меню:** Користувачам доступний широкий вибір страв, які поділені на зрозумілі категорії (Бургери, Піца, Суші, Салати). Інформація про кожну страву включає якісне зображення, детальний опис інгредієнтів та ціну.
* **Швидка фільтрація та пошук:** Зручна система фільтрів дозволяє в один клік відсортувати меню за бажаною категорією або знайти конкретну страву за назвою.
* **Інтерактивний кошик:** Додавання товарів до кошика відбувається миттєво. Користувач може легко змінювати кількість порцій, переглядати загальну кількість замовлення та видаляти зайві позиції.
* **Особистий профіль:** Система підтримує авторизацію користувачів, дозволяючи зберігати особисті дані та керувати своїм профілем.
* **Адаптивний дизайн:** Інтерфейс автоматично підлаштовується під будь-який розмір екрана. Сайт виглядає та працює однаково добре як на великих десктопних моніторах, так і на екранах мобільних телефонів.

## Дизайн та інтерфейс
Візуальна складова Ctrl+Eat базується на принципах мінімалізму. Використано світлу тему оформлення з акцентним фірмовим зеленим кольором, що асоціюється зі свіжістю та швидкістю. 

Особливістю дизайну є легка стилізація під IT-тематику (назва "Ctrl+Eat", елементи у вигляді коду), що робить бренд упізнаваним.

## Технологічний стек
Незважаючи на легкість інтерфейсу, під капотом проєкту працюють сучасні вебтехнології:
* **React** — для побудови інтерактивних компонентів користувацького інтерфейсу.
* **React Router** — для забезпечення безшовної навігації.
* **Tailwind CSS** — для створення унікального та адаптивного дизайну.
* **Vite** — для максимальної швидкості роботи та збірки проєкту.
